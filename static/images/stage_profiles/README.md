# Stage Profile Images

Place stage elevation profile images in this folder.

## Naming Convention

Files must be named exactly as follows:
- `stage_1_profile.png` (or `.jpg`)
- `stage_2_profile.png`
- `stage_3_profile.png`
... and so on through `stage_20_profile.png`

## File Format

- Accepted formats: PNG, JPG, JPEG
- Recommended width: 800-1200px
- Images will be displayed full-width on the stage detail page

## How It Works

These images will automatically appear on each stage's detail page under the "Stage Profile" section, below the stage description.

If an image is missing, the profile section will be hidden automatically.
